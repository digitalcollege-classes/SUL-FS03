# A12 :: Organização das Atividades

### Conteúdo

Trabalhar e desenvolver as atividades de acordo com os conhecimentos adquiridos ao longo do curso

### Objetivo

Utilizar o trello para organizar o projeto em raias:  
\- Product Backlog  
\- Sprint Backlog  
\- TODO  
\- DOING  
\- DONE
